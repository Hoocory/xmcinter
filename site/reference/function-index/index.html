    Function Index

    This index lists the important public functions documented for notebook analysis. Lower-level implementation helpers are intentionally omitted unless they are useful in advanced workflows.

    xmcinter.xmcfiles

    • merge_output() - merge numbered XMC text output files into one DataFrame.
    • read_parnames() - read parameter names from parameters.txt.
    • read_spectra() - read spectrum FITS products into histogram tuples.
    • fake_deconvolution() - write a DataFrame back to deconvolution-file format.
    • remove_nans() - drop rows with missing values.

    xmcinter.wrangle

    • filterblobs() - filter rows by parameter ranges.
    • simplefilterblobs() - single-column filtering helper.
    • filtercircle() - filter or weight blobs by circular regions.
    • weighted_median() - weighted median statistic.
    • weighted_std() - weighted standard deviation.
    • weighted_modes() - weighted modal estimates.
    • credible_region() - Bayesian credible region helper.
    • make_histogram() - histogram helper with optional weights and errors.
    • normalize_histogram() - normalize histogram values.
    • iter_err() - estimate per-iteration statistic variation.
    • make_spectrum() - create a spectrum from selected blobs.
    • normclean() - advanced normalization-based blob cleaning.
    • line_emissivities()
    • line emissivity helper.
    • blob_line_photons()
    • line photon estimate helper.

    xmcinter.xmcmap

    • make_map() - create FITS maps and return map arrays from blob parameters.
    • movie_from_stack() - write movie frames from an image stack.

    xmcinter.diagnostics

    • check() - run the high-level diagnostic workflow.
    • clean() - filter iterations and add derived blob quantities.

    xmcinter.plots

    • chi2() - reduced chi-square convergence plot.
    • trace() - parameter trace plots.
    • histogram() - single weighted/unweighted histogram.
    • histogram_grid() - grid of posterior histograms.
    • scatter() - single parameter scatter plot.
    • scatter_grid() - scatter matrix for blob parameters.
    • standard_spectra() - standard data/model spectrum plot.
    • spectra() - plot supplied spectrum histogram tuples.
    • spectrum_from_blobs() - create and plot a spectrum from a blob subset.
    • plot_lines() - add emission-line annotations to an existing Bokeh spectrum figure.
    • errorbar() - add symmetric error bars to a Bokeh figure.

    xmcinter.astro_utilities

    • norm_to_em() - convert norm to emission measure.
    • em_to_density() - estimate density from emission measure and volume.
    • em_to_mass()
    • estimate mass from emission measure and volume.
    • em_to_mass2()
    • mass estimate using blob size and distance.
    • convert_distance() - distance unit conversion.
    • convert_arcsec() - angular-to-physical size conversion.
    • angstroms2keV()
    • wavelength/energy conversion.
    • K2keV() - temperature conversion.
    • gaussian_volume() - Gaussian blob volume.
    • distance() - 2-D coordinate distance.
    • xmc2wcs() - convert XMC coordinates to sky coordinates.
    • wcs2xmc() - convert sky coordinates to XMC coordinates.
    • normalize_image() - normalize a FITS image.
    • stack_images() - sum aligned FITS images.
    • transfer_header() - copy a FITS header.
    • xspec_abund_to_nomoto()
    • abundance ratio conversion.
    • xspec_abund_to_nomoto_dataframe()
    • abundance ratio conversion on DataFrames.
    • fetch_lines()
    • fetch X-ray line list rows.
    • show_xray_lines()
    • inspect X-ray line list rows.
    • redshift_line()
    • redshift a line energy or wavelength.
    • distance2redshift()
    • approximate distance-to-redshift helper.

    xmcinter.file_utilities

    • files_to_list() - list files matching a pattern.
    • ls_to_list() - legacy shell-based file listing.
    • parse_file_line() - parse a tokenized file line.
    • read_list() - read a text list.
    • file_lines() - count/read file lines.
    • fetch_file() - find a file by pattern.
